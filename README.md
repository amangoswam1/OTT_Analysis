# Executive Summary

This project demonstrates an end-to-end Product Analytics workflow for an OTT (Over-the-Top) streaming platform.

Using SQL, business intelligence dashboards, and product analytics frameworks, the project analyzes user login behavior, engagement patterns, retention, churn, promotional effectiveness, and platform performance to generate actionable business insights.

The objective is to simulate how a Product Analytics or Business Intelligence team supports strategic decision-making using data.

---

# Business Problem

OTT platforms generate millions of user interactions every day.

Understanding why users return, disengage, churn, or respond to promotions is essential for maximizing customer lifetime value.

This project aims to answer questions such as:

- Which acquisition channels attract the highest-quality users?
- Which user cohorts exhibit the highest retention?
- What drives customer churn?
- Which promotional campaigns deliver measurable business impact?
- How do engagement patterns differ across countries and age groups?
